Introduction

So how do we start a game? First of all, we must include all of the javascript source files, like so:

<script>if(!ingenioJS){ var ingenioJS = {}; }</script>
<script src="ingenioJS/lib/dom.js"></script>
<script src="ingenioJS/lib/toolkit.js"></script>

<script src="ingenioJS/cache.js"></script>
<script src="ingenioJS/engine.js"></script>
<script src="ingenioJS/engine.audio.js"></script>
<script src="ingenioJS/engine.hitmap.js"></script>
<script src="ingenioJS/engine.hitmap.pathfinder.js"></script>
<script src="ingenioJS/engine.controllers.item.js"></script>
<script src="ingenioJS/engine.controllers.message.js"></script>
<script src="ingenioJS/engine.controllers.player.js"></script>
<script src="ingenioJS/engine.controllers.quest.js"></script>
<script src="ingenioJS/compositor.js"></script>
<script src="ingenioJS/renderer.js"></script>
<script src="ingenioJS/renderer.animation.js"></script>
Next we must design a level. We do this by opening up THE EDITOR

In there you will notice a lot of things down the side panel. These are things that can be put into the level. Where does it get these items from? The clue is when we look into the Javascript in the editor.html file. When we create the editor object, we pass it as the parameters something like this:

models: {
  index: 'game/models/forest.json',
  base: 'game/assets/forest',
},

The first part, the json file, is a description of all the different assets that will be used to create the level. It is a description that tells the engine what size each sprite is, whether it can be walked over or not, etc. It also contains a link to the image file that will be used to display it. These image file paths are all relative to the second line, which is the "base" part.

To familiariase yourself with this, why don't you create a new png file and put it in the "base" folder. It can even go in a sub directory if you want. Then add the description of this into the json file and reload the editor. You should see the png file you made appear in the side bar on the left.

Once you have created a level, you can export it. Copy the JSON that gets produced and save it to the assets directory.
